# RPB0003: global.json not found or missing SDK version

## Description

This warning occurs when a global.json file is not found, or when the global.json file does not specify an SDK version.
In this scenario, the latest installed SDK is used, which can lead to inconsistent and non-reproducible builds.

## Cause

This warning appears when:
- No global.json file exists in the project directory or any parent directory
- A global.json file exists but does not contain an SDK version specification

## Resolution

Create a global.json file in your repository root with an SDK version specification. You can pin your current SDK
version using the .NET CLI:

```bash
dotnet new globaljson
```
